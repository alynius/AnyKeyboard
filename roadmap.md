# Roadmap

This roadmap outlines completed, in-progress, and planned improvements.

---

## Completed

### Core Features
- Browser-based virtual keyboard with 100+ language layouts
- Mobile-friendly responsive UI
- Language-specific pages with SEO optimization
- Typing speed test with real-time WPM & accuracy
- Random sentence typing practice (78 languages)
- Phonetic typing for Arabic, Russian, Hindi, Greek, and more
- Name transliteration across 30+ scripts
- Alt-code utility pages (32 languages)
- Blog with keyboard and language guides

### April 2026 — Full Product Audit
- **Security hardening:** 0 npm vulnerabilities, CSP tightened, SRI on analytics, API authentication
- **Performance optimization:** ~600 KB less per page load, browser caching, ISR, font optimization, per-language typing data
- **Code quality overhaul:** -15,000 lines removed, locale variants consolidated, modular JS architecture, ESLint + Prettier
- **UI/UX improvements:** RTL support for 10 languages, OS theme detection, loading skeletons, CSS custom properties system
- **Accessibility compliance:** Skip navigation, ARIA labels on all keyboard keys, focus management, reduced motion support, WCAG 2.4 coverage

---

## In Progress

- Additional keyboard layouts for underserved languages
- Expanding phonetic typing coverage to more languages

---

## Planned

- Browser extension for typing in any text field
- Offline/PWA support
- Advanced typing statistics and progress tracking

---

This roadmap evolves based on user feedback and usage data.
